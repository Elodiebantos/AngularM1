Bienvenue sur le site de rendu des devoirs

Ce site est destiné aux étudiants de MIAGE en M1 et M2.

Le site de rendu des devoirs est un espace de travail qui permet de tenir une liste des devoirs qu'ils auront à rendre au cours du semestre, activer un rappel, éditer ou déposer un devoir.

Description du site

Cette page est référencée dans le code Angular comme étant le component "accueil". Chaque élément qui apparaît dans le sidenav représente un component en angular. L'élément "Liste des devoirs" est celui qui regroupe la plupart des TP en Angular : il gère l'affichage, l'addition et la suppression de devoirs.

Vous pouvez vous référer au fichier README du github pour plus d'informations.
